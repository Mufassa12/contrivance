│ DISCOVERY WORKFLOW                                                                 │
└────────────────────────────────────────────────────────────────────────────────────┘

  1. CLICK DISCOVERY MENU
     └─ Navigate to /discovery

  2. SELECT ACCOUNT
     ├─ Search Salesforce accounts
     ├─ Autocomplete suggestions appear
     └─ Select target account

  3. ANSWER DISCOVERY QUESTIONS
     ├─ Tab 1: Security & Compliance
     ├─ Tab 2: Infrastructure & Cloud
     ├─ Tab 3: Development & DevOps
     └─ Tab 4: Data & Analytics

  4. SAVE RESPONSES
     └─ Click "Save Discovery Responses"

│ TESTING INSTRUCTIONS                                                               │

  1. Open http://localhost:3000
  2. Login with your credentials
  3. Click "Discovery" in the sidebar menu
  4. Search for and select a Salesforce account
  5. Navigate through each technology vertical tab
  6. Try different question types:
     - Check/uncheck boxes (Security frameworks, Cloud providers)
     - Select radio buttons (Data classification, RTO)
     - Type in text fields (Security tools, Monitoring solutions)
  7. Click "Save Discovery Responses"
  8. Verify success message appears
  9. Test "Clear Responses" button
